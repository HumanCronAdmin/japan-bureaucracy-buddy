# Japan Bureaucracy Buddy

Free interactive checklist for foreigners navigating Japan's post-arrival bureaucracy — filtered by visa type.

**Live:** https://humancronadmin.github.io/japan-bureaucracy-buddy/

## Features
- Visa-type filter (5 types: Permanent Resident, Engineer, Student, Specified Skilled, Spouse)
- Step-by-step flow: Address Registration → My Number → Bank Account → Health Insurance → Pension
- Bank comparison table (6 major banks)
- 2027 insurance rule alert
- Progress tracking with localStorage
- Mobile-first design

## Part of JapanDevTools
One of 14+ free tools for foreigners living in Japan. See all tools at [JapanDevTools](https://github.com/HumanCronAdmin).
